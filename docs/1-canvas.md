# Canvas — The New Visual Page Builder

Drupal CMS 2.0 introduces Canvas as the default editing experience, providing true drag-and-drop with live preview and real-time editing. Instead of toggling between back-end forms and preview screens, editors can now build and edit pages directly in the browser.
We'll take a deeper dive into the custom-built Mercury component library that provides common building blocks like cards, testimonials, heroes, menus, and accordions.
