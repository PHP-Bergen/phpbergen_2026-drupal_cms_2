# Mercury and Byte Theme



## Byte
Byte is preconfigured as a marketing site for a SaaS-based product, with an elegant dark design, along with a blog, newsletter signup, pricing pages, and a contact form. Byte is the first of what will become a broader template marketplace, and it showcases how Drupal CMS 2.0 eliminates the blank-slate problem. Everything in Byte is built with Canvas, making it a perfect showcase for the new editing experience in action.

* Byte is a "site template", ie built with recipies included.
* Complete with content types.


## Mercury 

The Mercury theme is a modern, component-based starter kit introduced as a key part of Drupal CMS 2.0. It is specifically designed to work in tandem with Drupal Canvas, the new default drag-and-drop visual editing experience. Unlike traditional themes that dictate a rigid look, Mercury serves as a flexible foundation for building high-performance, accessible websites.

* Starter kit, meant for forking
* Based on Tailwind
* Rich component library
* WCAG AA
* Ready for Canvas

How to clone Mercury:

php core/scripts/drupal generate-theme my_custom_theme --name "My Custom Theme" --description "A custom theme based on Mercury." --starterkit mercury
